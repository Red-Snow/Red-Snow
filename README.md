### 🛡️ PhantomStrike — AI-Powered MCP Pentesting Framework

> Connect any MCP-compatible AI agent to a full **Kali Linux environment** and run natural-language-driven penetration tests with 600+ tools.

```
You:    "Scan 192.168.1.1 for open ports, then check all web services for SQLi."
Claude: Runs nmap → finds port 80/443 → runs nikto + sqlmap → returns a full report.
```

| Feature | Detail |
|:--------|:-------|
| 🤖 **AI Clients** | Claude Desktop · Cursor · VS Code Copilot · Gemini CLI |
| 🛠️ **Structured Plugins** | 12 tools with parsed JSON output |
| 🐚 **Universal Shell** | `run_kali_shell` — any Kali command, interpreted by AI |
| 🌐 **Deployment** | All-in-Kali · Split (Host + VM) · Docker |
| 🔗 **Protocol** | Model Context Protocol (MCP) — stdio + HTTP |
| 🗄️ **Storage** | SQLite scan history · async job queue · FastAPI backend |

**Supported Tools:**

| Category | Tools |
|:---------|:------|
| 🌐 Network | `nmap` · `rustscan` · `masscan` |
| 🕸️ Web App | `nuclei` · `gobuster` · `sqlmap` · `ffuf` · `nikto` |
| 🔍 OSINT | `subfinder` · `amass` |
| 🔑 Password | `hydra` |
| ☁️ Cloud | `trivy` |
